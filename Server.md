##  딥러닝 서버 구현

###  구현 기능
- 애플리케이션과 통신하여 데이터 송•수신, 의상 피팅 및 헤어 피팅 수행

<br>

###  구현 내용
-	애플리케이션과 통신하여 이미지 및 텍스트 데이터를 송•수신
-	애플리케이션에서 가상 피팅 요청 시 데이터를 전달받아 AI 모델을 통해 피팅 수행
-	가상 피팅 결과 이미지를 애플리케이션에 전송
-	원활한 통신을 위해 base64를 활용하여 이미지를 텍스트로 변환

<br>
